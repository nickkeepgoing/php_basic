# 🐘 PHP Fundamental Lab

คลังเก็บไฟล์ซอร์สโค้ดนี้คือพื้นที่รวบรวมการเรียนรู้และฝึกฝนพื้นฐานภาษา PHP ของผม ซึ่งถูกจัดทำขึ้นเพื่อใช้เป็นหลักฐานการทำความเข้าใจโครงสร้างภาษา (Core Syntax) และตรรกะการเขียนโปรแกรม (Programming Logic) ก่อนจะก้าวเข้าสู่การพัฒนาเว็บแอปพลิเคชันที่ซับซ้อนขึ้น

## 📋 เกี่ยวกับ Repository นี้
โปรเจกต์นี้ไม่ได้ถูกสร้างขึ้นมาเพื่อเป็นแอปพลิเคชันที่พร้อมใช้งาน แต่เป็น **"ห้องทดลอง"** ที่เก็บรวบรวมแบบฝึกหัดและการพิสูจน์แนวคิด (Proof of Concepts) ในหัวข้อต่างๆ เช่น:
- **Core PHP:** การจัดการตัวแปร, ชนิดข้อมูล และการควบคุมทิศทางของโปรแกรม (Control Structures)
- **Data Processing:** การจัดการ Array, String และการสร้างฟังก์ชันเพื่อใช้งานซ้ำ
- **Web Interaction:** การทำงานร่วมกับ HTTP Request (GET/POST), Sessions และการจัดการฟอร์ม
- **Persistence:** พื้นฐานการเชื่อมต่อและจัดการฐานข้อมูล (CRUD operations)

## 💡 สำหรับผู้เข้าชม (Portfolio Reviewers)
ในฐานะผู้พัฒนา ผมให้ความสำคัญกับ **"พื้นฐานที่แข็งแกร่ง"** โค้ดที่ปรากฏในนี้สะท้อนถึง:
1. **การเรียนรู้อย่างเป็นระบบ:** แสดงลำดับการทำความเข้าใจจากหัวข้อที่ง่ายไปสู่หัวข้อที่ซับซ้อน
2. **ความเข้าใจในไวยากรณ์:** การเลือกใช้เครื่องมือที่เหมาะสมในภาษา PHP 8.x
3. **การจัดระเบียบโค้ด:** ความพยายามในการเขียนโค้ดที่อ่านง่ายและมีการแยกส่วน (Folder Structure) ตามหัวข้อการเรียนรู้

## 🛠 Tech Stack & Environment
- **Language:** PHP 8.x
- **Development Environment:** PHP Built-in Server / Environment ที่รองรับ PHP
- **Focus:** Server-side Logic, Web Fundamentals
