# 🐘 PHP Learning Journey (Basic)

ยินดีต้อนรับสู่คลังเก็บความรู้และแบบฝึกหัดในการเรียนรู้ภาษา PHP ขั้นพื้นฐานของผมครับ Repo นี้สร้างขึ้นเพื่อบันทึกความก้าวหน้า สรุปเนื้อหา และเก็บโค้ดตัวอย่างที่ได้ฝึกฝน

## 🎯 เป้าหมายการเรียนรู้
- เข้าใจโครงสร้างและไวยากรณ์พื้นฐานของ PHP
- ฝึกฝนการเขียนโปรแกรมแบบ Logic และการจัดการข้อมูล
- สร้างพื้นฐานที่แข็งแรงก่อนต่อยอดไปยัง Framework (เช่น Laravel) หรือการสร้างโปรเจกต์จริง

## 📚 หัวข้อที่กำลังศึกษา (Roadmap)

### 🟢 Level 1: พื้นฐานสุดๆ (The Basics)
- [ ] **Introduction:** การติดตั้ง environment (XAMPP/Docker) และการเขียน Hello World
- [ ] **Variables & Data Types:** การประกาศตัวแปร, String, Integer, Float, Boolean
- [ ] **Operators:** การคำนวณ (+, -, *, /) และการเปรียบเทียบ
- [ ] **String Manipulation:** การจัดการข้อความและฟังก์ชันที่เกี่ยวข้อง

### 🟡 Level 2: Logic & Control Flow
- [ ] **Conditionals:** การใช้ `if`, `else`, `elseif` และ `switch case`
- [ ] **Loops:** การวนซ้ำด้วย `for`, `while`, `do-while` และ `foreach`
- [ ] **Functions:** การสร้างฟังก์ชัน, การรับค่า (Parameters) และการคืนค่า (Return)

### 🟠 Level 3: การจัดการข้อมูล (Data Handling)
- [ ] **Arrays:** Indexed Array, Associative Array และ Multi-dimensional Array
- [ ] **Superglobals:** การใช้งาน `$_GET`, `$_POST`, `$_SESSION`, `$_COOKIE`
- [ ] **Form Handling:** การรับข้อมูลจากฟอร์มและการ Validation เบื้องต้น

### 🔴 Level 4: การทำงานกับระบบภายนอก
- [ ] **File System:** การอ่าน/เขียนไฟล์ (Read/Write Files)
- [ ] **Database (PDO/MySQLi):** พื้นฐานการเชื่อมต่อและสั่งงานฐานข้อมูล (CRUD)
- [ ] **Error Handling:** การใช้ `try...catch` และการแก้ Bug

## 🛠 เครื่องมือที่ใช้
- **Language:** PHP 8.x
- **Server:** PHP Built-in Server / XAMPP
- **Editor:** VS Code

## 📂 วิธีการใช้งาน Repo นี้
โค้ดตัวอย่างจะถูกแบ่งเป็นโฟลเดอร์ตามหัวข้อที่เรียนรู้ คุณสามารถเข้าไปดูในแต่ละโฟลเดอร์เพื่อศึกษาโค้ดและคำอธิบาย (ถ้ามี) ได้เลยครับ
